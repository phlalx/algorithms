# Big integers

## Addition

```
def sum(num1, num2):
    n = max(len(num1), len(num2))
    num1 = num1.zfill(n)
    num2 = num2.zfill(n)
    carry = 0
    res = []
    for d1, d2 in zip(reversed(num1), reversed(num2)):
        x = int(d1) + int(d2) + carry
        d = x % 10
        carry = x // 10
        res.append(str(d))
    if carry:
        res.append(str(carry))
    print(res)
    return "".join(reversed(res))
```
